1.0.5 Documentation

Simbrain combines three interacting components--a neural network, a world, and a set of one or more gauges. These components can be used to simulate the way a creature represents and navigates its environment. The network simulates the creature's brain; the world simulates the creature's environment; the gauges plot arbitrary subspaces of the network's neurons and weights. Simbrain aims to be as easy-to-use and as self-explanatory as possible. There is much you can accomplish by experimentation. For documentation on each component, click below, or try this quick start to immediately begin creating simulations with Simbrain. Click here for more on the design philosophy of Simbrain.

Workspace
The workspace component of Simbrain contains both network and world. It is the space within which to create multiple networks, worlds and gauges to be tested. Each workspace can be though of as an individual experiment in how neural networks work.


Networks
The network component of Simbrain represents a simulated neural circuit. The network component can be used on its own to build and analyze neural networks or it can be used to control the creature in the world component.


Worlds
The world component of Simbrain simulates the environment of the creature whose "brain" is represented by the network. Using the world and the network you can study "perception" only, behavior only, or sensorimotor coordination. For example, to study "perception" only, you can manually push the creature around and see how the network responds to different objects.

Gauges
Perhaps the most unique aspect of Simbrain is its interaction with HiSee, a tool for projecting high-dimensional data to two dimensions. Arbitrarily many gauges can be created, each of which represents an arbitrary dimensional subspace of the network's weights and nodes. The patterns which emerge in these gauges show how the network of a creature represents its environment.
