DataWorld

Data World is a spreadsheet styled data bank that has a multitude of different uses. Each row in the Data World symbolizes one vector with length equal to the amount of columns. Much like the stimulus values of an Odor World, a sensory neuron can only couple to one of the values in a row of information. Information can be added to the Data World spreadsheet through severeal means. If a neuron has an Output Command established with a Data World, its activation is stored in that neuron's specified column. Information can also be directly entered into the Data World spreadsheet, or even, once it has been saved, in a spreadsheet editor that can read and modify *.csv files (such as Excel).

Features of Data World

Size: Data World can have any number of Rows and Columns.

Data World can be Randomized with any bounds.

Iteration Mode changes the current Data World row for each iteration. Iteration Mode can be used to train networks and store the activations of neurons.

Send buttons push data from a Data World to coupled networks and iterate those networks. This is useful when one wants to test the impact of different input vectors on a network. Rather than selecting different rows then iterating the networks, a 2-step operation (times however many coupled networks there are), the user just presses the send buttons.
