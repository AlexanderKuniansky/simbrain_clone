Couplings

Sensors and motor commands simply correspond to column numbers in the table.

If a neuron is connected to a column as sensor, then when the neuron is updated it pulls the value from that column of the currently selected row.

If a neuron is connected to a column as motor command, then when the neuron is updated it sets the value for that column of the currently selected row.
