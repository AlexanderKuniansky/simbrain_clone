Coupling

Coupling to a Data World is a pretty mechanical relationship; a number is sent or received by a neuron when a network is coupled to a Data World. In a Data World, information is stored in vectors, or rows, where the interaction between the neuron and the World occurs in one specific dimension of those vectors. When establishing a coupling with a Data World, one specifies an index, or column, for the neuron to affect or be affected by. If a neuron is an Input Sensor for a particular column, what value that neuron takes on is determined by which row is highlighted in the Data World.

For example, if a neuron was an Input Sensor for column number 5, and the third row of Data World was selected, then when the network is iterated, the neuron would take on the value in the third row and fifth column of the Data World. Likewise, if the neuron was giving Output Commands to column number 5, and the third row of the Data World was selected, then the third row and fifth column of the Data World would take on the neuron's activation when the network is iterated.
