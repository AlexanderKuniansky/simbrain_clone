Simbrain > World

 OdorWorld
The Odor World component of simbrain is used for creating real-to-life worlds for inputting stimulus values into networks. Every object, including agents, can have a stimulus value. Each object represents a stimulus which can be set to be detected by an agent.
DataWorld
The Data World component of simbrain is similar in usage as to Odor world. The only difference is that instead of using an agent as a stimulus detector, stimulus values are directly sent to neurons. This method of stimulus input is an easy, fast way to input data that doesn't need to utilize any kind of decay function.
