Subnetworks

Subnetworks are a special kind of group used when special update rules are required that go beyond what is available using the standard update method for "loose" collections of neurons and synapses,  and to facilitate special types of weight update, e.g. via supervised learning. 

For more information on specific types of subnetwork click on the items in the menu on the right.

Creating Subnetworks

To create a subnetwork use Insert > Insert network > ... either from the network menu or by right clicking inside the network panel.
