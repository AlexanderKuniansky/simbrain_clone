Simbrain > Network > Common Topics and Glossary

Clipping

This will clip the activation level to the value of upper bound if greater than upper bound or lower bound if lower than lower bound.

Randomization

Is a uniform distribution within the upper and lower bounds of relevant components.

Synapse

Also "weights." This represents... We can talk about the presynaptic signal and postsynaptic signal or source and target neurons.

Spiking Neurons

These neurons produce discrete "firing" events.

Time vs. Iteration

If the network consists entirely of discrete neuron types, time is shown in iterations. Otherwise it is shown using the current time-step, which must be the same through the network.
