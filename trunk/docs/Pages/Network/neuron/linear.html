This is a common type of activation function in neural networks. The activation level a of a this type of node is a linear function of the weighted inputs W:

A piecewise linear function resembling a sigmoidal function can be created by making use of clipping (see below).

If slope is set to 1 and bias to 0, linear neurons simply pass on their weighted inputs. This is especially useful for input and output nodes.

To set the horizontal or vertical intercept to some specific value see the bias term below.

Bias

A fixed amount of input to this node, denoted by b above.

To set the horizontal intercept to a value h set b = -h.

To set the vertical intercept to a value v set b = v/m.

Slope

The slope of the linear activation function, denoted m above. The slope scales the value of the weighted inputs plus the bias b.

Use Clipping

See clipping.

Add Noise

If this is set to true, random values are added to the activation of the node after all the linear activation function is applied. The type of added noise can be specified by the fields of the noise generator interface.
