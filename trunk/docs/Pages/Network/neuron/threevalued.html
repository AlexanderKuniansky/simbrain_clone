Three Valued

This is just like a binary neuron except it takes three values and has two thresholds. If the the weighted input W plus a bias term b is greater than an upper threshold,then it takes on an upper value. If it is less than a lower threshold, it takes on the lower value. Otherwise it takes on a middle value.

 

Bias

A fixed amount of input to this node.

Lower Threshold

Below this value activation is the lower value.

Upper Threshold

Above this value activation is the upper value.

Lower Value

Value neuron takes when weighted inputs are lower than lower threshold.

Middle Value

Value neuron takes when weighted inputs are between lower and upper thresholds.

Upper Value

Value neuron takes when weighted inputs are above upper threshold.
