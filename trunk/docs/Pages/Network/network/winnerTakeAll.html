Winner-Take-All Network

Function

The neuron with the highest weighted input in a winner-take-all network takes on an upper value, all other neurons take on the lower value. In case of a tie the node which wins is arbitrary (the first in an internally maintained list).

 

Parameters

Lower Value

The activation of any node which does not have the greatest weighted input.

Upper Value

The activation of the node which has the greatest weighted input.

 

Initialization

When creating a WTA network you can specify the lower and upper values, as above, and also how many neurons are in the network. The standard layout options are also avaialble.

 

Number of Units

The number of units in the network.
