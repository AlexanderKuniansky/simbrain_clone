Setting Synapse Properties

Setting single synapse: double click on a synapse or right click on it and select set properties. Since synapses are relatively small, you can either zoom in or lasso the synapse of interest and select Network > Set Properties.

Setting multiple neurons: select more than one synapse, and either double click on one of the synapse, or select set properties from the popup or network menu. A multiple synapse dialog box will appear. If the selected neurons are of different types, only the common properties appear. Otherwise all properties appear. Consistent properties are shown, inconsistent properties are marked by a null string: ". . ." Changes you make will apply to all selected synapses.
