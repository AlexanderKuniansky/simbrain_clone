Neurons

Neurons or "nodes" are represented by circles. The numbers inside the neurons represent their "activation level" (we can think of the absolute value of this number as representing the rate of a firing of a neuron). These values are rounded off; the exact values can be determined by lingering over a neuron or double clicking on it. The color represents whether the activation is greater than 0 (red), less than 0 (blue), or equal to 0 (white). These color conventions can be adjusted in the preference dialog. The intensity of the color corrsponds to how far this level is between the upper or lower bound of the neuron and 0.

There are a variety of different neuron types in Simbrain, each of which has numerous parameters which can be set. Properties common to all neurons are described on this page. Properties specific to particular neurons on described on their own pages.

Setting Neuron Properties

Neuron parameters can be adjusted as follows:

Setting single neurons:

Double click on a neuron or right click on it and select set properties.

Setting multiple neurons:

Select more than one neuron, and either double click on one of the neurons, or select set properties from the popup or network menu. A multiple neuron dialog box will appear. If the selected neurons are of different types, only the common properties appear. Otherwise all properties appear. Consistent properties are shown, inconsistent properties are marked by a null string: ". . ." Changes you make will apply to all selected neurons.

Common Neuron Properties

Some properties of neurons are common across neuron types.

Activation

The current level of activity of this neuron. How this should be interpreted (as firing rate, voltage potential, or something else, depends on the neuron type). This is reflected graphically as the color of the corresponding node and is shown in the node text. The color and the range of values it takes can be set in network properties.

Increment

This sets the amount that a neuron is incremented when it is manually adjusted. For example, if increment is set to .1, then each time the up arrow is pressed the neuron will increase its activation by .1.

Upper bound

This field plays different roles depending on the type of the neuron (documented accordingly) but in general it determines the maximum level of activity of a node. It also determines the range of colors which a neuron can take on. This and lower bound determine the bounds of randomization.

Lower bound

This field plays different roles depending on the type of the neuron but in general it determines the minimum level of activity of a node. It also determines the range of colors which a neuron can take on.
