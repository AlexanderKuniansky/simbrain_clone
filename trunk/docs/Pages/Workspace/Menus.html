Simbrain > Workspace > Menus

File

Open Workspace Opens a configuration of workspace components (i.e. a configuration of networks, worlds, and gauges)

Save Workspace/Save Workspace As Saves a configuration of workspace components. The file that is saved and opened just consists of a path to the workspace components, and must be somewhere in the Simbrain directory.

Import Workspace/Export Workspace Saves a configuration of workspace components in a form that is easily moved from one computer to another. When exporting, this creates a directory with the name you specify, copies of all workspace component files (e.g. network files, world files, etc.), and a workspace file with the same name. When importing opening up the directory and open the file with the same name. For example, if you export a workspace named "Test" a directory named Test will be created. To import the same workspace open the Test directory and open the file "Test.xml."

Clear Workspace closes all open workspace components.

New Network creates a new Network.

New Odor World creates a new OdorWorld.

New DataWorld creates a new DataWorld.

New Gauge create a new Gauge.
