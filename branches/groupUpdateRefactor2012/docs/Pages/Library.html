Using simbrain as a library (Beta)


Simbrain can be used as a library in a separate java program.   To do this, you must get Simbrain from source and then use ant to build the jar files of interest. Currently network (no gui), network with gui are supported.   That is,

> ant networkjar
> ant networkguijar



To get a feel for how to use the Simbrain API,  try opening up different scripts in (for example) the scriptmenu directory.

There is not yet a workspace only jar.


Applets

Applets can be deployed using the network.jar.
