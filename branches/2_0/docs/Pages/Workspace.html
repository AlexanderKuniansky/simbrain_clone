Workspace

The workspace is the frame in which all networks, worlds and gauges are created, saved, and opened. Note that the linkages between components are mostly handled within the components themselves, in particular the network component. A workspace can be thought of as an individual neural network simulation, experiment, or lesson. The workspace is visible mainly through the menu bar. Components may be opened or created through commands in the File and Insert dropdown menu.
