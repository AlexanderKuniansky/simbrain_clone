Simbrain > DataWorld

DataWorld is used for inputting data, similar to an Odorworld, in a much easier way.

Inputting data into DataWorld

Inputting data into DataWorld is as simple as entering data into a spreadsheet. The values entered into the the DataWorld become the level of activation of the attached neuron. Pressing send multiple times increments the activation of that neuron by the value sent to it.

Connecting neurons to the DataWorld

Attaching the DataWorld to network is similar to attaching an OdorWorld to a network. Each column can be used to increment or decrement the activation level of a neuron each time the "send" button is pressed. To attach a column to a neuron, simply right click on the neuron to be attached, go down the popup menu to sensors, then follow the sub-menus to the DataWorld and finally to the column number to attach the neuron to.

Menu Items

File contains commands for opening, saving and closing data worlds.
