Lab: Basic Features of Neural Networks

1. Compare distributed and local representations. Create a world with four objects. Two of the objects should have distributed representation, two should have local representations. Make a five dimensional network for them to respond to. Export the workspace and email it to me.

Create a new odor world with an agent and four objects.

  • Start by clearing the workspace and creating a new odor world.
  • Add an object to the world using the right mouse click and selecting "Add new object" from the context menu.
  • Repeat until the world has four objects within it.
  • Change each object to a different image using the mouse right click and selecting set object properties from the context menu. Select the image to be used for the object from the drop down list at the top of the object properties window. The image itself has no bearing on how the object works, it is only to make them easier to distinguish from one another.
  • Arrange the objects so they have adequate space between them. Placing them near the four corners of the world frame is recommended.
  • Add an agent to the world using the right mouse click and selecting "Add new agent" from the context menu.

Your world should look like this.

Modify object settings.

  • Right click on any of the four objects within the world and select Set object properties from the context menu.
  • Change the Stimulus values as follows.
  • For two of the objects, make each of the first five rows 0 except one, which should have a positive value. These are local representations.
  • For the other two objects, make each of the first five rows have a different value. These are distributed representations.

Create a network containing five nodes.

  • Add a new network to the workspace.
  • Create four neurons.
  • Set each of the neurons' input to center by right clicking on one of the neurons. Highlight and follow the path Sensors, Odor World 1, Mouse 1, Center, 1.
  • Repeat for the other four setting each neuron to a different number. When finished, each neuron should have an input arrow with center and the number underneath it. Each neuron should have a unique number 1-5.

Observe local and distributed representations.

  • Within the world, move the mouse to one of the objects.
  • Note network reaction.
  • Repeat with other three objects.
