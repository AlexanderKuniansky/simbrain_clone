Lab: A (Modified) Braitenberg Vehicle

In this example we couple a neural network to a simple environment. The network below controls the little mouse on the right. The mouse is similar to a vehicle described by the neuroscientist Valentino Braitenberg, in his book, Vehicles. The vehicle occurs in chapter 2 of that book, and is described as being "aggressive," because it runs up to the thing it's attracted to as if it's attacking it. In this case the mouse will attack the fish.

 

Again begin by clearing the workspace and opening a sample file:

  • Clear the workspace by using File > Clear Workspace located in the workspace window.
  • Open a workspace using File > Open Workspace.
  • Select lessons > attract.xml.

Play with the mouse:

  • To the right of a network is a "world," in this case an "odor world."
  • Drag the mouse around within the world.
  • Note how depending on whether the mouse is to the left or right of the fish, more or less activation will be on the left or right input node.
  • Note how the direction the mouse turns depends on which output node, left or right, has more activation.

Play with the fish:

  • Next move the fish image the same way the mouse was moved within the world frame and note the network reaction.
  • Finally, press the play button within the network frame.
  • Move the fish around in the world as indicted above.

This example uses couplings to send and receive information between the network and world frames which will be covered in more detail later on.

Exercises

1. Modify the attract network so that it always turns left.

2. Modify the attract network so that it avoids objects.

3. Create another object which the network ignores.

4. Create a network which is attracted to swiss cheese and avoids fish.
