Glossary

Activation Space: The set of possible activation vectors for a neural network.

Activation Vector: A vector describing the activation values of the nodes of a neural network.

Artificial neural network: A collection of interconnected units which processes information in a roughly brain-like way

Attractor: a state or set of states A with the property that if you are in a nearby state the system will always go towards A.

Axon: The part of the neuron that carries outgoing signals to other neurons.

Basin of attraction: the set of all states that tend towards a given attractor.

Biological neural network: A set of interconnected neurons in an animal brain

Cognitive science: The interdisciplinary study of mind.

Computational neuroscience: The study of the brain using computer models.

Connectionism: The study of psychological phenomena using artificial neural networks.

Dendrite: The part of the neuron that receives signals from other neurons.

Dynamical system: A rule which tells us what state a system will be in at any future time, given any initial condition.

Initial Condition: the state a dynamical system begins in.

Invariant set: a collection of orbits.

Mathematical Model: An abstract representation of a real process.

Neuron: a cell in the nervous system specialized to transmit information.

Orbit: the set of states that are visited by a dynamical system from an initial condition.

Phase portrait: A picture of a state space with important orbits draw in it.

Projection: A way of representing a group of points in a high-dimensional space in a lower dimensional space.

Repellor: a state or set of states R with the property that if you are in a nearby state the system will always go away from R

Simplifying assumptions: Features of a phenomenon being modeled which are not captured by the model itself.

Soma: The cell body of a neuron.

State Space: the set of possible states of a system.

State: A specification of values for all the variables describing a system. The state of a neural network is typically an activation vector.

Synapse: The junction between nerve cells where a nerve impulse is transferred from one neuron to another.

Vector Valued Function: A function which takes vectors as inputs and produces vectors as outputs.

Vector: An ordered list of numbers. (note that this is a definition for our purposes only; it is not a mathematical definition)

Weight Space: The set of possible weight vectors for a given neural networks.

Weight Vector: A vector describing the strengths of the weights of a neural network.
