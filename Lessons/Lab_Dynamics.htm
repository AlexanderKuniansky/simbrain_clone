Lab: Dynamical Systems in Neural Network Theory

Make a network with four nodes, fully interconnected. To do this:

1) Create four nodes by clicking the new neuron button four times.
2) Connect the four nodes to themselves and the other four nodes. Do this by (1) selecting all the nodes and (2) right click on a target node and select connect. Repeat the process for each successive target node.
3) Randomize the weights by selecting all weights using "W" (or Edit > Select > Select All Weights), and either pressing the randomize button or pressing "R"

Now look for attractors and limit cycles. You do this by

1) Selecting all neurons using the keyboard command "N"
2) Randomizing these neurons by pressing pressing the randomize button or using keyboard command "R"

Be sure to check for the zero vector! You do this by selecting all neurons and pressing "C" or the clear button.

Now, once you've done all this mail me your network and answer the following questons:

1) How many dimensions are in the network's activation space?
2) How many dimensions is the network's weight space?
3) How many attractors did you find?
4) How many repellors did you find?
5) How many limit cycles did you find?
6) For each limit cycle, say what its period is.

Another thing you can do is open a gauge while you perform these experiments to watch what happens in the networks' activation space.
