Raster

Can be used in 2 modes, activation or spiking mode.

Activation Mode

The horizontal axis is time steps (like time series plot). The vertical axis shows which values occur in the network being plotted. If a network's activation vector is all 0's, a few 1's, and a .5, then 0, .5, and 1 are shown in the raster plot.

Spiking Mode

In spiking mode, this is used with a neuron group spike train attribute. See Coupling Networks. The vertical axis shows the indices of neurons that have spiked. It only advances when some neurons have spiked.

Component Panel

Add: Add a times series to the plot.

Remove: Remove a times from the plot.

Clear Graph Data: Clears all data from the plot.

Show Graph Properties: Opens up properties menu for the plot.

Menus

Share the same menu properties as the time series plot. See Menus.

Right-Click Menu

The context menu is a JFreeChart menu.
