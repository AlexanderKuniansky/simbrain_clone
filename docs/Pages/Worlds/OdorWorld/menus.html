Menu

File menu contains commands to save, open, close worlds and to set general world properties.

Edit menu contains commands to cut, copy and paste entities within worlds as well as clear all items from the world componet.

Scripts menu allows for the loading, running and stopping of scripts used within the world componet.

Help menu accesses the help portion of the world component.

 

Popup menus

Popup menus are accessed by right-clicking or control-left clicking in the world component.

Clicking on an object. Each object has a popup menu associated to it. To get the popup menu for an object place the cursor over top of the object's icon and right click. Users can use the object's popup menu to cut, copy, paste, or delete an object in the world component, bring up the entity dialog box, or bring up the world dialog box.

Clicking outside any object. To get the popup menu for the world component place the cursor in the world component's window but outside of any objects in the world and right click. Users can use the world's popup menu to either add a new object, agent, or wall to the world componentor or bring up the world dialog box.
