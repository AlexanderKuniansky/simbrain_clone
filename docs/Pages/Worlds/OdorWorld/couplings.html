Agents

 

Odor worlds are coupled to networks via agents. Agents are shown (currently) as mice in OdorWorld windows:

Agents can move in various directions and sense objects in their surroundings. Note that agents can smell one another, but cannot smell themselves.

The Odor World coupling menus are separated by agents, so that indvidual neurons can be coupled to specific creatures in the world. This allows, for example, for multiple creatures controlled by separate neural networks to interact.

Motor commands

North, South,... These move the creature in absolute directions.

Left, Right, Forward, Backward. These move the creature relative to its current orientation.

Sensors

The sensory menus are broken down according to the location of a "whisker" (even though these are suppsoed to be olfactory stimuli the whisker metaphor helps), and then a number which corresponds to a particular stimulus dimension.

Left, Right, Center: The Location of the sensor on the agent. These can be thought of as left and right whiskers (whose properties can be set in the agent dialog), and a central nose.

1,2,3...: The numbers correspond to indices of sensory receptors. The value associated with a given index depends on what values are associated with it in different objects (where those values are set in the entitity dialog).

Manually moving agents

 

Agents and objects can be dragged directly. Agents can also be used using keyboard commands. Click on the agent you wish to move. This "selects" the agent, though there is currently no indication that the agent has been selected in the GUI. The selected agent can now be moved forward, reverse, left and right by pressing the arrow keys on the keyboard.
