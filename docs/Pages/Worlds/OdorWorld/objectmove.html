Moving objects

Objects can be moved by dragging. Place the cursor over the object's icon, hold down the left mouse button, and move the computer mouse. The object's icon will move with the cursor until you let up on the left mouse button. The sensory inputs to the network component can either be updated as the object is dragged or only when the user lets up on the left mouse. This option is set in the world dialog box. (Note: It is possible to move an object's icon outside of the world component window. Enlarging the world window may reveal the object.)
