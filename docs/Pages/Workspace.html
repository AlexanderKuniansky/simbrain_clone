Workspace

Workspace is the frame in which all OdorWorlds, DataWorlds, Gauges, and Networks are created. Everything within a workspace is its own investigation of how neural networks operate.

Menus

File is where workspaces can be opened, saved, created and cleared.

  • Open Workspace command executes the file chooser dialog for opening of saved workspace configurations.
  • Save Workspace/Save Workspace As allows the user to save workspaces that have been created during the current session
  • Clear Workspace closes all open Network, OdorWorld, DataWorld and Gauge frames that are currently open within the workspace frame and prepairs the workspace for new simulations
  • New Network creates a new Network frame for creation of new neural nets.
  • New World creates a new OdorWorld frame for creating interacion environments.
  • New DataWorld creates a new DataWorld frame for inputting world stimulations in a easier fashion.
  • New Gauge create a new Gauge used for graphing different network data points.
