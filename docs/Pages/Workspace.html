Simbrain > Workspace

Description

The workspace is the frame in which all networks, worlds and gauges are created, saved, and opened. Note that the linkages between components are mostly handled within the components themselves, in particular the network component. A workspace can be thought of as an individual neural network simulation, experiment, or lesson. The workspace is visible mainly through the menu bar. The file menu can be used to add new components.

Menus

File

Open Workspace opes a configuration of workspace components (i.e. a configuraton of networks, worlds, and gauges)

Save Workspace/Save Workspace As Saves a configuration of workspace components.

Clear Workspace closes all open workspace components.

New Network creates a new Network .

New Odor World creates a new OdorWorld.

New DataWorld creates a new DataWorld.

New Gauge create a new Gauge.

Help calls up this help page.

Examples

Open an existing workspace using file > open workspace, move and resize the windows slightly, save using file > save workspace, and reopen to observe the changes.

Select clear workspace to remove all current components from the workspace, create a single network using new network, open a sample network, and open a gauge using the gauge button in the network frame. Run the network using its play button and observe the pattern which results in the gauge.
