Workspace Examples

Open an existing workspace using file > open workspace, move and resize the windows slightly, save using file > save workspace, and reopen to observe the changes.

Select clear workspace to remove all current components from the workspace, create a single network using new network, open a sample network, and open a gauge using the gauge button in the network frame. Run the network using its play button and observe the pattern which results in the gauge.
