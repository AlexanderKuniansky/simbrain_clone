Simbrain > Quick Start

Run a simulation

When Simbrain launches, simply press play in the network toolbar and observe activity propagating through the network and the creature moving. Open different workspaces using File > Open Workspace and press play in the network windows to see what happens.

Manually moving objects in Worlds

Open the simulation attract.xml. Clikc the play button then move the object in the world around and observe the creature in the world chasing after it. Now try repulse.xml.

Finding stable states

Open the workspace lessons > iac.xml. While in the network component, select all nodes with "N", press "R" to randomize the nodes and press the space bar a few times to update the network until it settles into a stable state (that is, until the nodes don't change when you press space). Repeat; press "R" and "space, space,space" repeatedly until the net settles in a stable state, then "R" and "space, space, space." again, etc. In this way get a sense of the stable states of that network.

Build a network

Open a new network window and create a few neurons using the button. Then connect them as described here. Then run the network. Or, create a network of a specific type using Network > New Network > ... Just press OK if a dialog box appears. and see how it behaves when you press run. You can connect this network to a world by selecting Workspace > New World and then right clicking on neurons and selecting Sensors > ... and Motor Commands ... from the popup menu.

Study network parameters

Select all nodes with "N", then double-click on a node. Change it's decay value, activation function, etc. Then move the creature around a bit (by left dragging) to see how network dynamics have changed.

Study learning

Select all weights with "W", then double-click on a weight. Change it's learning rule or other parameters. Then run the network and see whether the weights change size.

Use Gauges

Open a gauge by pressing the add gauge button and open any workspace. If there is an odor world drag the creature around the different objects and watch the patterns that emerge in the gauge. Try opening different worlds and gauges and seeing what patterns show up in the gauge. When a pattern has developed for a period of time, switch the gauge projection chooser from "coordinate" to "PCA" and from "PCA" to "Sammon." When it is in Sammon press the play button inside that gauge to see what patterns emerge.
