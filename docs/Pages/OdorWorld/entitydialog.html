Entity Dialog Box

The entity dialog box allows the user to set the properties of objects in the world component. The figure below shows what a typical identity dialog box looks like. To open the entity dialog box for an object, right click on the object to get the object's popup menu and select "Set object Properties" (or alternitively, double click on the object). To accept changes made, click on "OK" at the bottom of the dialog box. To reject changes to the object's properties click on "Cancel" at the bottom of the dialog box.

 

Image name: The image name allows the user to select the icon they would like to use to represent the object.

Entity: Name of entity. Can be any name not being used by any other entity within current world. Entities from different worlds can have the same name.

Agent tab:

  • Whisker angle is the angle at which stimulus is detected. The angle is measured from the head of the mouse being 0 degrees, counterclockwise. The opposide side of the mouse is found by mirrioring the angle from the opposite side of the agent.
  • Whisker length is a mesure of how far away from the agent a stimulus can be dectected.
  • Turn increment represents how many degrees the agent will turn.
  • Straight movement increment is how many units the agent will move with each movement increment.

Miscellaneous tab:

  • Edible objects can be eaten by agents.
  • Bites to die is how long it will take for the item to be eaten by the agent.
  • Resurection probability is the likelyhood an edible item will re-appear at the next iteration.

Stimulus values tab:

  • Stimulus demensions shows how many stimulus to use.
  • Stimulus Vectors are associated with "distal stimulus vectors." Distal stimulus vectors can be thought of as a profile of the effect an object will have on the cureature's receptors if it is near it. The distal stimulus vecors of all the objects are transformed on the basis of distance betwen them and the creature and the presence of noise. These transformed stimulus vecors are then combined to create the input vector that will be sent to the neural network.
  • Randomize stimulus creates random numbers for all stimulus vectors within the upper and lower limits indicated.

Stimulus dispersion tab:

  • Decay function is a way of determining how the stimulus will be detected from different distances away from the agent.
    • Step is the simplest decay function where the stimulus has a constant value within a given radius and no value outside the radius.
    • Linear decay causes the stimulus intensity to decrease linearly with distance until it reaches zero. Beyond that distance the intensity remains zero. Dispersion is the linear decay function.
    • Gaussian decay causes the stimulus intensity to decrease according to the Gaussian function (a bell shaped surface). This is a good representation of how substances diffuse through fluids like air and water. Dispersion is the Gaussian standard deviation.
    • Quadratic decay actually causes the stimulus intensity to decrease according to the inverse square law i.e. as the reciprocal of the square of the distance. This is a good representation for how the intensity of light and sound deceases with distance. Dispersion is the rate of decay of the quadratic function.
  • Dispersion is a measure of how rapidly the stimulus intensity decays for each of decay functions. Its precise meaning depends on the particular decay function that has been selected. In each case the larger the value of Dispersion the more slowly the stimulus intensity decreases with distance.
  • Add noise: The "Add Noise" option allows users to add noise to the sensory inputs produced by objects in the World component. The perception of the real world by living organisms often involves some element of randomness or noise. This noise can come from the environment or it can be intrinsic to the sense organs themselves. When the "Add Noise" option is selected each stimulus value of the object has a small random number generated for it. The random numbers are added to the stimulus values which are then sent to the input nodes of the Network component. Note that the value of each input node is restricted to lie between a preset minimum and maximum value. Stimulus values which are out of bounds for the input node are clipped to the nearest boundary value. The value of the input node can also be rounded off.
  • Noise level allows the user to control the value of the standard deviation for the Gaussian distribution. The numbers along the slider stand for percents and the value of "Noise level" ranges from 0% to 100%. The standard deviation is set equal to the chosen percentage of the maximum stimulus value. When the value of Noise level is 0% no noise is added. When the value of Noise level is 100% the amount of noise which is added at each iteration is on the same scale as the largest stimulus value. Values of Noise level in between 0% and 100% provide intermediate amounts of noise to the stimulus values.
