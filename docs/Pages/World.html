Simbrain > World

World components represent the "world" of a neural network. They produce inputs to and accept outputs from networks comonents. They represent an environment and a set of sensory and / or motor transducers. In general they convert a given type of sensory information (visual patterns, smells, sounds) into a set of values which is sent to the input nodes of network, and conversely, convert output vectors into motor information (motion, sounds, text, etc.) These couplings between neuronns and worlds are handled using the "coupling" interface). OdorWorld
The Odor World component simulates olfactotion (smell) and motion in a 2-d world. The creature or "agent" smells objects and can react to them by turning and moving. Each object is associated with a vector of values, representing a pattern of activity over an agents olfactory receptors, which scaled according to how close an agent is to that object. The objects can also be "eaten."

DataWorld
The Data World component is the smiplest world type, which simply sends values to and accepts values from a network. Each row reprsents an input or output vector.
