Common Topics and Glossary

Clipping

If a neuron or weight uses clipping, its activation value or strength is modified as follows. If its activation value or srength is above the upper bound for that neuron or weight, it is set to that upper bound. If it is below the lower bound for that neuron or weight, it is set to that lower bound.

Noise Generator

Some simbrain components allow noise to be added to their value. Noise is generated using the following fields:

Distribution: The probability density function that the random values follow. Currently Gaussian or uniform.

Upper bound: If the distribution is uniform, this represents the upper bound of the distribution. If the distribution is Gaussian and clipping is being used, this cuts off the upper tail of the distribution.

Lower bound: If the distribution is uniform, this represents the lower bound of the distribution. If the distribution is Gaussian and clipping is being used, this cuts off the lower tail of the distribution.

Mean value: This is the mean of the Gaussian distribution.

Standard deviation: This is the standard deviation of the Gaussian distribution.

Use clipping: If true, upper and lower bounds are used as described above.

 

Randomization

Is a uniform distribution within the upper and lower bounds of relevant components.

Synapse

The inputs to a node can have different degrees of influence on that node. This is measured a number often called a "weight." Since these are connections between "neurons" they are also called synapses.

We can talk about the presynaptic signal and postsynaptic signal or source and target neurons.

Spiking Neurons

These neurons produce discrete "firing" events.

Time step vs. Iteration

The time step is a global variable which is used whenever differential equations are numerically integrated. Generally speaking, the smaller the time-step, the more accurate the numerical integration.

Several neuron and synapse types are defined in terms of differential equations. We call these "continuous" neurons and synapses. When a continuous neuron or synapse is used, an iteration of the network represents an interval of time whose length equals the time-step. This is indicated by the word "iterations" being replaced with "seconds" in the time-step indicator at the bottom-left of the network panel.
