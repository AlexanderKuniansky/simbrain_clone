Setting Synapse Properties

Setting a single synapse: double click on a synapse or right click on it and select set properties. Since synapses are relatively small, you can either zoom in or lasso the synapse of interest and select Network > Set Properties.

Setting multiple synapses : select more than one synapse, and either double click on one of the synapse, or select set properties from the popup or network menu. A multiple synapse dialog box will appear. If the selected neurons are of different types, only the common properties appear. Otherwise all properties appear. Consistent properties are shown, inconsistent properties are marked by a null string: ". . ." Changes you make will apply to all selected synapses.

Common Synapse Properties

These are properties common to all synapses, regardless of type. All of them have some reflection in the graphical interface, and some of them only matter to the interface (e.g. increment). For some synapse upper bound and lower bound does not play any role, but does limit how larget the graphical representation of the synapse can be.

Strength: how "strong" this synapse is. This is represented by the size of the weight. By default a blue weight represents an inhibitory synapse with value less than 0. A red weight represents an excitatory synapse with value of 0 or greater. These colors and the size ranges can be set in network preferences. Strength can be set by the up and down arrows and can be randoamized.

Increment: this sets the amount that a synapse is incremented when it is manually adjusted. For example, if increment is set to .1, then each time the up arrow is pressed the synapse will increase its activation by .1.

Upper Bound : This field plays different roles depending on the type of the synapse (documented accordingly) but in general it determines the maximum strength of a weight. It also determines the range of colors which a neuron can take on. This and lower bound determine the bounds of randomization.

Lower Bound : This field plays different roles depending on the type of the synapse (documented accordingly) but in general it determines the maximum strength of a weight. It also determines the range of colors which a neuron can take on. This and lower bound determine the bounds of randomization.

Delay: A positive integer, which delays how many iterations or time-steps must occur before the pre-synaptic signal (the activation of the source neuron) is delivered.
