ToolbarS

Toolbars can be turned off using the network view menu.  

Edit Toolbar

This toolbar allows you to set the edit mode for the network, visible as the cursor graphic.

Zoom in or out in the network panel.  By pressing z the one can toggle between zoom in and zoom out mode. When this tool is active, the cursor will change to a small magnifying glass with either a "+" or a "-" in the center representing the active zoom mode.  Clicking and lassoing a region of the screen can be used to zoom in to that region of the screen .  Note: Zooming and panning are only effective if autozoom is turned off. 
Pan Mode. Move the view of the network around (to actually move the network select all and drag the network). Note: Zooming and panning are only functional if autozoom is turned off.
Selection mode. Select nodes and weights by lassoing or left-clicking.
Text mode.  In this mode text can be added to the simulation.
Wand mode.  Inject activation to nodes quickly for testing.  More information can be found in the wand tool instructions at the bottom of the page.

Data Toolbar 

This toolbar is for running the network, independently of the workspace.   It is hidden by default (because it can be easily confused with workspace run / step) and must be turned on using the view menu.  

Run network . Repeatedly iterates the network. Note that the image on the button becomes that of a "stop" button . Pressing this button will stop the network. Thus the play/stop button acts as a toggle switch.
Step through one update cycle (Also keyboard space and S, depending on interaction mode).

Edit Toolbar 

This toolbar allows you to create and delete neurons, and change their state.

Set all selected nodes (weights and or nodes) to a random value (Also keyboard R). Randomization is a uniform distribution between the upper and lower bounds set for each neuron or weights.
Set all selected neurons to 0. Weights are ignored. (Also keyboard C)
Create a neuron at last position clicked on screen. (Also keyboard N)
Delete currently selected neurons. (Also keyboard delete or backspace).

Clamp Toolbar 

These are toolbar buttons, which can be on or off, to control whether clamping is enabled for neurons and weights.  

Clamp neurons. See clamping.
Clamp weights. See clamping.

Tool Details:


Wand Tool:

The wand tool, which can be found on the network toolbar, is a tool used to alter the activations of neurons within its range. As of the current version of Simbrain the wand only injects activation. The addiion of multiple modes of neuron alteration to the wands function are planned for future releases.

To use the wand simply:
    1) Click on the wand icon
    2) Pass the wand over neurons while holding the mouse key
    3) All neurons will reach their maximum level (the “upper bound” value), as shown below.


The radius in pixels of the wand's effect can be adjusted by going to Edit > Wand/Edit Mode Preferences within a network window.

Text Tool:

The text tool can be used to place a text object anywhere in the network panel. While it provides little in the way of functionality it does allow the user to label functional items within the network panel. Upon its addition to the panel, the text object can be moved, cut, pasted, and copied like any other network object.
