Neuron

To adjust one neuron or a group of neurons, double click on a neuron or right click and select set properties. A dialog will appear. You can also directly edit the xml files.

There following properties are common to all neuron types:

Activation : the current level of activity of this neuron. This is reflected graphically as the color of the corresponding node.

Increment : this sets the amount that a neuron is incremented when it is manually adjusted. For example, if increment is set to .1, then each time the up arrow is pressed the neuron will increase its activation by .1.

Upper bound : This field plays different roles depending on the type of the neuron (as noted in relevant documentation) but in general it determines the maximum level of activity of a node. It also determines the range of colors which a neuron can take on.
