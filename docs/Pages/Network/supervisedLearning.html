Supervised Learning


Trainers are used to implement supervised learning in Simbrain.  They are like teachers.   You can use trainers to "teach" a network to associate associate input patterns with output patterns. You are basically telling the network, when you get this input, produce this output, for a whole bunch of inputs and outputs.

To use a trainer in simbrain you first create a network that has a trainer associated with it.  Currently there are two
such networks:Backprop and Least Mean Squared (LMS). 

Backprop : work with arbitrarily many layers.

Least mean square (LMS): works with 2 layer networks.  Useful becaues it can be much faster than backprop.
