Adding Text

To insert text into the document viewer, simply insert your text into the edit tab of the document viewer, keeping the tags <html>, <body>, </html>, and </body> where they are. The text can be stylized, for instance, using the tag <b></b> bolds the text. For more HTML formatting, click here.


Text (HTML format) can also be imported from your local drive. To do this, use the format below to access the file. docs is the folder in which is where the file is located in, follow by the name of the file itself SimbrainDocs.html.


Example of a local link: <a href = "file:docs/SimbrainDocs.html">docs</a>
