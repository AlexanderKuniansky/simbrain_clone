Adding Image

An image can be inserted into the document viewer from your local drive (or an external source). To insert an image from your local drive, simply use the following format shown below: docs is the main folder in which is where the image file is located, Images is a sub-folder of docs, and simbrainlogo.gif is the the file itself.


Example of a local image: <img src = "file:docs/Images/simbrainlogo.gif">
